# SPEC.md — Project Specification

> **Status**: `FINALIZED`

## Vision
To build a Qiskit-based quantum simulation toolkit that models the time evolution of a 1D Quantum Harmonic Oscillator using the Linear Combination of Unitaries (LCU) algorithm. The project will bridge the gap between continuous physical systems and quantum circuits by automatically handling discretization, Hamiltonian Pauli decomposition, truncated Taylor series expansion for the time evolution operator ($e^{-iHt}$), and execution via LCU on a local Qiskit statevector simulator.

## Goals
1. **System Discretization & Hamiltonian Generation:** Automatically map the continuous 1D Harmonic Oscillator Hamiltonian into a finite-dimensional matrix based on a user-defined number of qubits (representing the truncated Hilbert space dimension $N=2^q$).
2. **Pauli Decomposition:** Provide robust methods to decompose the target Hamiltonian matrix into a sum of Pauli strings ($H = \sum c_i P_i$).
3. **Time Evolution via LCU:** Implement a truncated Taylor series approximation of $e^{-iHt}$ and construct the corresponding Qiskit LCU quantum circuit (involving state preparation of ancillas, multiplexed target unitaries, and post-selection/measurement).
4. **End-to-End Simulation Pipeline:** Execute the simulated time evolution using Qiskit's local statevector simulator, allowing users to input the number of qubits ($q$), evolution time ($t$), and Taylor expansion truncation order ($K$).

## Non-Goals (Out of Scope)
- Simulation of systems with noise or execution on real quantum hardware.
- Simulation of 2D or 3D Harmonic Oscillators (restricted to 1D).
- Alternative quantum simulation algorithms (e.g., Trotterization, qDRIFT) beyond LCU.
- Global installation; all dependencies and execution must be strictly isolated to a local Python virtual environment.

## Users
Physicists, quantum algorithm researchers, and students who want an end-to-end sandbox to experiment with the Linear Combination of Unitaries (LCU) method for time-evolving continuous quantum systems.

## Constraints
- **Technical constraints:** Must use Qiskit and be executable on a local Python virtual environment (`venv`).
- **Mathematical constraints:** The Taylor expansion and spatial discretization must remain computationally feasible for local simulation (likely capped at a moderate number of qubits, e.g., $q \leq 8-10$, due to local statevector limits).

## Success Criteria
- [ ] A script successfully generates a Pauli string decomposition for a 1D Harmonic Oscillator given an arbitrary qubit count $q$.
- [ ] The theoretical matrix $e^{-iHt}$ matches the unitary operator generated by the truncated Taylor series up to an expected error bound based on order $K$.
- [ ] The final Qiskit LCU circuit executes on a statevector simulator without errors.
- [ ] The project runs entirely within an isolated `venv` with a functional `requirements.txt`.
